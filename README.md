# Optics Lab — 光学シミュレーション

高校物理「光」単元の授業向け Web シミュレーション集。
ブラウザだけで動作し、インストール不要です。

**デモサイト**: https://phys-ken.github.io/Optics/

---

## シミュレーション一覧

| シミュレーション | 概要 |
|---|---|
| レンズ（点光源） | 凸レンズによる点光源の像 |
| 物体の像 | 大きさのある物体の実像・虚像 |
| 平面鏡 | 反射の法則と虚像 |
| 凹面鏡 | 凹面鏡の焦点と像 |
| 光線モデル | スリットを通る光の直進性 |
| 目のしくみ | 水晶体・網膜とピント調節 |
| 可変絞りレンズ | 絞りと収差の関係 |
| 3D光線モデル | Three.jsによる立体光線 |
| 練習問題 | レンズの式計算問題自動生成 |

## 機能

- **3モード切替**: 自習 / 授業（教卓投影用） / 作図（印刷用）
- **白背景キャンバス**: グレースケール印刷に対応
- **リアルタイム描画**: Canvas 2D / Three.js によるインタラクティブシミュレーション
- **レスポンシブ**: PC・タブレット・スマホ対応

## ファイル構成

```
index.html          ランディングページ（ハブ）
css/main.css        グローバルスタイル
js/optics.js        共通ライブラリ（物理計算＋描画＋UI）
sims/               各シミュレーションページ
test/run-tests.py   構造テスト
_archived/          旧バージョン（2020年版）
```

## ローカルでの実行

```
python -m http.server 8765
```

ブラウザで http://localhost:8765 を開く
